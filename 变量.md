```
Dish
├── Production (1:1)
├── Storage (1:n)
├── Transport (1:n)
├── Supplier (n:n)
│
Company
├── Dish (1:n)
├── Employee (1:n)
│
User
│
Merchant
│
Recall
│
Handover
```
1. Dish表: traceCode字段建立唯一索引
2. Production表: dishId字段建立索引
3. Storage表: dishId和storageWarehouse字段建立复合索引
4. Transport表: dishId和transportBatchNumber字段建立复合索引
5. Employee表: name和department字段建立复合索引
